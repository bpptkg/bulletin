=====================
Bulletin Web Services
=====================

Bulletin web services is a project that aim to make WebObs MC3 bulletin,
SeisComP, and BPPTKG seismic bulletin database in sync. The services listen for
update event in the WebObs, e.g. when the observer try to pick or modify an
event, and do calculation necessary to update seismic bulletin database.
